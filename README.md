## Getting Started

1. **Use this template** by clicking "Use this template" above, or clone directly:
   ```sh
   git clone <your-repo-url> my-app
   cd my-app
   ```

2. **Install dependencies:**
   ```sh
   pnpm install
   ```

3. **Set up your database:**
   ```sh
   pnpm migrate
   ```

4. **Start developing:**
   ```sh
   pnpm dev
   ```
